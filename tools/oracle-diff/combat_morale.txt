# Combat damage oracle (morale / spirit): 気力・精神コマンドを変えながらダメージ式を
# cross-engine で突合する。両エンジンとも中立地形 (C#=EmptyTerrain→地上 / Rust=env 陸) に
# 配置し、実データの地形適応 (S/A/B/C/D) を素直に効かせて素のダメージ式を比較する
# (combat_damage.txt と同方式 — @option は使わず @terrain も指定しない)。
#
# 指令:
#   @morale <unitName> <value>   そのユニットのメイン気力を value にする (corpus 順に累積)。
#                                C# は SetMorale で [50,150] にクランプするため 50..150 を使う。
#   @spirit <unitName> <name>    そのユニットに精神 <name> をアクティブにする (累積)。
#                                C# = Unit.MakeSpecialPowerInEffect (SPDList データ駆動の効果レベル)、
#                                Rust = status slice へ名前付与 (名前ベースの固定倍率)。
#
# このフィクスチャ (スパロボ戦記) の sp.txt 定義 (system/sp.txt):
#   熱血 → ダメージ増加Lv10  → C# 1+0.1*10 = ×2.0   (Rust 熱血=×2)
#   魂   → ダメージ増加Lv15  → C# 1+0.1*15 = ×2.5   (Rust 魂=×3  ← 乖離が予期される)
#   気合 → 気力増加Lv1 (即効) → 気力 +10 相当 / ダメージ増加効果なし
#   鉄壁 → 被ダメージ低下Lv7.5 (防御側) → C# 被ダメージ低下 (1/4 相当) (Rust 鉄壁=÷4)
#
# 実在ユニット: マジンガーＺ / マジンカイザー / アフロダイＡ / 機械獣ガラダＫ７ (人工知能)。
# 武器 idx は combat_damage.txt の既知 valid index を流用: マジンガーＺ 3=光子力ビーム(格)。

@unit マジンガーＺ 0 味方 人工知能 10
@unit マジンカイザー 0 敵 人工知能 10

# === 気力スケーリング =====================================================
# baseline: 双方 気力 100 (×1.0)
@predict マジンガーＺ マジンカイザー 3 ダメージ
# 攻撃側 気力 120 (攻撃力 ×1.2)
@morale マジンガーＺ 120
@predict マジンガーＺ マジンカイザー 3 ダメージ
# 攻撃側 気力 150 (攻撃力 ×1.5)
@morale マジンガーＺ 150
@predict マジンガーＺ マジンカイザー 3 ダメージ
# 防御側も 気力 150 (装甲 ×1.5)
@morale マジンカイザー 150
@predict マジンガーＺ マジンカイザー 3 ダメージ
# 攻撃側 気力 50 (攻撃力 ×0.5)、防御側 気力 50 (装甲 ×0.5)
@morale マジンガーＺ 50
@morale マジンカイザー 50
@predict マジンガーＺ マジンカイザー 3 ダメージ

# === 精神コマンド =========================================================
# 気力をリセット (双方 100) してから精神を乗せる
@morale マジンガーＺ 100
@morale マジンカイザー 100
# 熱血 (攻撃側) → ダメージ ×2
@spirit マジンガーＺ 熱血
@predict マジンガーＺ マジンカイザー 3 ダメージ
# 熱血 → 魂 (攻撃側)。C# は 熱血/魂 が同時付与でも MaxDbl で大きい方 (魂 Lv15=×2.5)。
# Rust は名前ベースで 熱血(×2) と 魂(×3) を別々に乗算する点に注意 (乖離が予期される)。
@spirit マジンガーＺ 魂
@predict マジンガーＺ マジンカイザー 3 ダメージ
# 鉄壁 (防御側) → 被ダメージ 1/4。熱血/魂 はそのまま (攻撃側)。
@spirit マジンカイザー 鉄壁
@predict マジンガーＺ マジンカイザー 3 ダメージ

# === 気合 (気力増加系 — ダメージ増加効果なし) =============================
# 別ペアで素の 気合 を確認 (攻撃側へ付与)。アフロダイＡ 1=格闘(格)。
@unit アフロダイＡ 0 味方 人工知能 20
@unit 機械獣ガラダＫ７ 0 敵 人工知能 20
# baseline
@predict アフロダイＡ 機械獣ガラダＫ７ 1 ダメージ
# 気合 (攻撃側): C# は ダメージ増加効果なし (気力 +10 は即時付与だが MakeSpecialPowerInEffect
# 経由では気力増加は適用されないため実質 no-op)。Rust は 気合=×1.2 を乗算する (乖離が予期される)。
@spirit アフロダイＡ 気合
@predict アフロダイＡ 機械獣ガラダＫ７ 1 ダメージ
